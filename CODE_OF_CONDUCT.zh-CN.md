# 贡献者公约行为准则

[English](CODE_OF_CONDUCT.md)

## 我们的承诺

作为成员、贡献者和管理者，我们承诺：无论年龄、体型、可见或不可见的残障、
族裔、性别特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、
外貌、种族、阶层、肤色、宗教或性身份与性取向如何，每个人参与社区都不会
受到骚扰。

我们承诺以有助于建设开放、友好、多元、包容和健康社区的方式行事与互动。

## 行为标准

有助于营造积极环境的行为包括：

- 对他人表现出同理心与善意；
- 尊重不同意见、观点和经历；
- 提供并坦然接受建设性反馈；
- 为错误负责，向受影响者道歉并从中学习；
- 关注整个社区的最佳利益，而不只关注个人。

不可接受的行为包括：

- 带有性意味的言语或图像，以及不受欢迎的性关注或追求；
- 恶意挑衅、侮辱、贬损评论以及人身或政治攻击；
- 公开或私下骚扰；
- 未经明确许可发布他人的住址、邮箱等私人信息；
- 发布 API Key、凭据、私人提示词或私人生成媒体；
- 其他在专业环境中可被合理认定为不当的行为。

## 管理责任

社区管理者负责解释并执行可接受行为标准，并会对其认为不当、威胁、冒犯或
有害的行为采取适当、公平的纠正措施。

社区管理者有权并有责任移除、编辑或拒绝不符合本准则的评论、提交、代码、 Wiki
修改、Issue 和其他贡献，并会在适当情况下说明管理决定的理由。

## 适用范围

本准则适用于所有社区空间，也适用于个人在公共空间正式代表社区的场合，例如
使用官方账户、通过官方社交媒体发言或作为指定代表参加活动。

## 执行

请通过仓库提供的私密联系/报告功能向维护者报告辱骂、骚扰或其他不可接受行为。
如果该功能不可用，请通过 GitHub 联系仓库所有者，不要公开事件细节。报告将被
及时、公平地审查。

所有社区管理者都必须尊重报告者的隐私与安全。

## 处理准则

社区管理者将依据以下准则确定后果：

### 1. 纠正

**社区影响：**使用不当语言或其他被认定为不专业、不受欢迎的行为。

**后果：**私下书面警告，解释违规性质，并可能要求公开道歉。

### 2. 警告

**社区影响：**单次事件或一系列行为构成违规。

**后果：**说明继续违规后果的警告，并可能在指定时间内禁止与相关人员互动。

### 3. 暂时封禁

**社区影响：**严重违规，包括持续的不当行为。

**后果：**在指定时间内暂时禁止与社区互动或公开沟通。

### 4. 永久封禁

**社区影响：**持续违规、骚扰个人、攻击或贬损某类人群。

**后果：**永久禁止在社区内公开互动。
